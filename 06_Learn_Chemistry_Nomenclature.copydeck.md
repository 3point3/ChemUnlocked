## 6.2 Ionic Compounds: Metal First, Nonmetal Second
- Paragraph 1: Many **ionic compounds** are made of a metal and a nonmetal. In these compounds, positive and negative ions attract each other.
- Paragraph 2: Later, you will also see ionic compounds that contain **polyatomic ions**, such as ammonium or sulfate. Start with the core pattern: metal first, nonmetal second. Once this is automatic, 6.4 adds polyatomic ions and 6.3 adds the Roman numeral step for variable-charge metals.
- Paragraph 3: **Start here with the two-step pattern:**
- Paragraph 4: Write the name of the metal exactly as it appears on the periodic table. Do not change it.
- Paragraph 5: Drop the end of the nonmetal's name and add **–ide**. For example: chlorine → chloride, oxygen → oxide, sulfur → sulfide, nitrogen → nitride, fluorine → fluoride, bromine → bromide, iodine → iodide.
- Paragraph 6: Here are the most common simple ionic compounds. Notice that none of the names use prefixes — the subscripts in the formula handle the ratio automatically.
- List item 1: Do **not** use prefixes like "di-" or "tri-" for ionic compounds.
- List item 2: The subscripts in the formula handle the ratios automatically.
- List item 3: Common mistake: seeing a subscript of 2 and trying to say "dichloride" in an ionic name. Do not do that.

## 6.3 Transition Metals: When You Need Roman Numerals
- Paragraph 1: Some metals can form more than one type of ion. These are mainly **transition metals** like iron (Fe), copper (Cu), and lead (Pb).
- Paragraph 2: When a metal can form more than one charge, you must add a **Roman numeral in parentheses** after the metal name. The Roman numeral shows the charge of the metal ion in that compound. The numeral comes from charge balance, not from the subscript by itself.
- List item 1: Metals that *always* form one charge, like Na+, K+, Mg2+, Al3+, and Ca2+, do **not** need Roman numerals.
- List item 2: Only use them when the metal can have more than one charge.
- Label 1: Pattern
- Label 2: Charge-balance snapshot

- Table row 2: FeCl2 | Fe2+ | iron(II) chloride
- Table row 3: FeCl3 | Fe3+ | iron(III) chloride
- Table row 4: CuO | Cu2+ | copper(II) oxide
- Table row 5: Cu2O | Cu+ | copper(I) oxide
- Table row 6: FeO | Fe2+ | iron(II) oxide

## 6.4 Polyatomic Ions: Keep the Ion Name
- Paragraph 1: A **polyatomic ion** is a group of atoms bonded together that carries a charge. It acts as one unit in a formula. These have to be memorized. There is no shortcut for the table itself — but the naming move is simple: if the ion is polyatomic, keep its memorized name exactly as given.
- Paragraph 2: Use this as your study reference. The ones you will see most often in this course are ammonium, hydroxide, nitrate, sulfate, carbonate, and phosphate — start there.
- Paragraph 3: When naming a compound with a polyatomic ion, use the name of the ion directly. Do **not** change it to "-ide."
- Paragraph 4: Examples: Ca(OH)2 = **calcium hydroxide** · NH4NO3 = **ammonium nitrate** · Na2SO4 = **sodium sulfate**

## 6.5 Molecular Compounds: Nonmetal + Nonmetal Means Prefixes
- Paragraph 1: A **molecular compound** is made of nonmetals bonded together. Nonmetals can bond in many different ratios, so we use **prefixes** to show exactly how many atoms of each element are in the molecule.
- Paragraph 2: These ten prefixes are the only ones you need for molecular compounds. The second element always gets a prefix; the first element skips "mono" when there is only one.
- Paragraph 3: If there is only 1 atom of the first element, skip "mono." If there are 2 or more, use the prefix that matches the subscript.
- Paragraph 4: Always use a prefix for the second element. Change the ending to –ide. If the prefix ends in a or o and the element name starts with a vowel, drop that vowel from the prefix (e.g., tetra- + oxide = tetroxide, not tetraoxide).

## 6.6 Binary Acids: No Oxygen, Use Hydro-
- Paragraph 1: In this unit, **acid names** are used when the compound is acting as an acid in water.
- Paragraph 2: A **binary acid** contains only two elements: hydrogen and one other nonmetal, with no oxygen. Start here with the fastest check: no oxygen means binary acid.
- Paragraph 3: All binary acids start with "hydro-"
- Paragraph 4: For chlorine → chlor, bromine → brom, iodine → iod, sulfur → sulfur, fluorine → fluor
- Label 1: Binary Acid Naming Pattern
- Table row 1: Formula | Name

## 6.8 Writing Formulas from Names
- Paragraph 1: Going name → formula is the harder direction because you construct the subscripts from scratch instead of reading them off a formula. The payoff is that the words in the name give you every clue you need — the name identifies the compound type, and the compound type tells you the formula-building rule.
- Paragraph 2: Once you identify the type, apply the matching formula-building rule:
- Paragraph 3: Write each ion with its charge. Find the smallest whole-number ratio that makes the total charge exactly zero. Those ratios are the subscripts — drop any subscript of 1.
- Paragraph 4: Example: *potassium oxide* → K+ and O2-. Need 2 K+ to cancel 1 O2- → **K2O**
- Paragraph 5: The Roman numeral is the metal's charge — use it directly. Write the metal ion with that charge, write the anion with its charge, then balance to zero for the subscripts.
- Paragraph 6: Example: *copper(II) oxide* → Cu2+ and O2-. Equal and opposite charges, 1:1 ratio → **CuO**
- Paragraph 7: Look up the polyatomic ion's formula and charge in the 6.4 table. Balance it against the cation charge to get subscripts. If the polyatomic ion needs a subscript greater than 1, enclose the whole ion in parentheses before adding the subscript.
- Paragraph 8: Example: *calcium nitrate* → Ca2+ and NO3-. Need 2 nitrate ions to cancel one calcium → **Ca(NO3)2**
- Paragraph 9: Each prefix tells you the subscript directly. First element: use the prefix as the subscript; no prefix means 1. Second element: use its prefix as the subscript, then drop the "–ide" ending to get the element symbol.
- Paragraph 10: Example: *dinitrogen monoxide* → di = 2 nitrogen, mono = 1 oxygen → **N2O**
- Paragraph 11: *Binary acid (starts with "hydro-"):* strip "hydro-" and "-ic acid," match the remaining root to its element, then write H + that element. Use charge balance for the subscript — most are 1:1 (HCl, HBr, HI, HF) except hydrosulfuric acid where sulfur is 2- so two H are needed.
- Paragraph 12: *Oxyacid (no "hydro-"):* –ic acid means the ion inside ends in –ate; –ous acid means the ion ends in –ite. Look up that polyatomic ion's formula and charge in the 6.4 table. Add enough H+ (each +1) to make the total charge zero.
- Paragraph 13: Example: *sulfuric acid* → –ic acid, no hydro- → sulfate = SO42-. Need 2 H+ to balance -2 → **H2SO4**
- Paragraph 14: Parentheses: when and why
- List item 1: Name ends in **"…acid"** → acid rules apply
- List item 2: Name contains a **Roman numeral** → ionic, variable-charge metal
- List item 3: Name contains a **polyatomic ion name** (sulfate, nitrate, hydroxide…) → ionic
- List item 4: Name has **prefix words** (di-, tri-, penta-…) → molecular
- List item 5: Metal name + nonmetal root ending in **–ide**, nothing else → simple ionic
- List item 6: Use parentheses only around a polyatomic ion that appears more than once.
- List item 7: Correct: Ca(NO3)2 — two nitrate groups, so they go in parentheses.
- List item 8: Correct: NaNO3 — only one nitrate group, so no parentheses.
- List item 9: Never put parentheses around a single-element ion like Cl or O.
- Table row 1: Name | Type signal | Formula
- Table row 2: potassium bromide | metal + –ide, no Roman numeral → simple ionic | KBr
- Table row 3: iron(III) chloride | Roman numeral III → Fe is 3+ | FeCl3
- Table row 4: calcium nitrate | nitrate is polyatomic → ionic | Ca(NO3)2
- Table row 5: dinitrogen monoxide | di- and mono- → molecular | N2O
- Table row 6: hydroiodic acid | hydro-…-ic acid → binary acid | HI
- Table row 7: nitric acid | –ic acid, no hydro- → nitrate inside → oxyacid | HNO3

# Examples Tab

## Example 1 · Naming a Simple Ionic Compound
- Paragraph 1: **Problem:** Name the ionic compound.
- Paragraph 2: **Given:** Formula = Na2S
- Paragraph 3: **Find:** Compound name
- Paragraph 4: The rule is metal + nonmetal = ionic compound. Na is a metal and S is a nonmetal, so Na2S follows ionic naming rules.
- Paragraph 5: Na is sodium. Sodium always forms +1, so no Roman numeral is needed.
- Paragraph 6: Sodium → sodium (unchanged). Sulfur → sulfide (–ide added).

## Example 2 · Naming an Ionic Compound with a Transition Metal
- Paragraph 1: **Problem:** Name the ionic compound and determine the Roman numeral.
- Paragraph 2: **Given:** Formula = Fe2O3; oxide = O2-
- Paragraph 3: **Find:** Compound name
- Paragraph 4: Because Fe is a transition metal, the naming rule is: find the metal charge from total charge balance before writing the Roman numeral. O is the oxide ion, O2-.
- Paragraph 5: Total negative charge: 3 oxide ions × (-2) = -6. Two iron atoms must supply +6 total. +6 ÷ 2 = **+3 per iron atom**. Roman numeral = III.
- Paragraph 6: Keep the metal name, add the Roman numeral, then write oxide: **iron(III) oxide**.

## Example 3 · Naming a Compound with a Polyatomic Ion
- Paragraph 1: **Problem:** Name the compound that contains a polyatomic ion.
- Paragraph 2: **Given:** Formula = Cr2(SO4)3; sulfate = SO42-
- Paragraph 3: **Find:** Compound name
- Paragraph 4: The rule is: keep the memorized polyatomic ion name, then use charge balance to find the transition-metal Roman numeral. SO42- is sulfate, and Cr is a variable-charge metal.
- Paragraph 5: 3 sulfate ions × (-2) = -6 total. 2 chromium atoms = +6 total. +6 ÷ 2 = **+3**. Roman numeral = III.
- Paragraph 6: Use the polyatomic ion name directly — do *not* change sulfate to sulfide. The compound is **chromium(III) sulfate**.

## Example 4 · Naming a Molecular Compound
- Paragraph 1: **Problem:** Name the molecular compound using prefixes.
- Paragraph 2: **Given:** Formula = N2O4
- Paragraph 3: **Find:** Compound name
- Paragraph 4: The rule is nonmetal + nonmetal = molecular naming with prefixes. N and O are both nonmetals, so the subscripts tell you which prefixes to use.
- Paragraph 5: 2 nitrogen atoms = **di**nitrogen
- Paragraph 6: 4 oxygen atoms = tetra- + oxide. "Tetraoxide" → drop the "a" → **tetroxide**. Add to name.

## Example 5 · Naming a Binary Acid
- Paragraph 1: **Problem:** Name the acid.
- Paragraph 2: **Given:** Formula = HBr(aq); no oxygen present
- Paragraph 3: **Find:** Acid name
- Paragraph 4: The rule is: if an acid in water contains no oxygen, name it as a binary acid with hydro- + root + -ic acid. HBr(aq) fits that pattern.
- Paragraph 5: Bromine → brom. Pattern gives: hydro + brom + ic acid = **hydrobromic acid**.

## Example 6 · Naming an Oxyacid
- Paragraph 1: **Problem:** Name the oxyacid from the ion ending.
- Paragraph 2: **Given:** Formula = H2SO3(aq); polyatomic ion = sulfite
- Paragraph 3: **Find:** Acid name
- Paragraph 4: The rule is: if an acid in water contains oxygen, name it from the polyatomic ion ending. H2SO3(aq) fits the oxyacid pattern.
- Paragraph 5: SO32- = sulfite (ends in –ite).
- Paragraph 6: Sulfite → sulfur + ous → sulfurous acid.

## Example 7 · Writing a Formula from an Ionic Name
- Paragraph 1: **Problem:** Write the formula for the ionic compound.
- Paragraph 2: **Given:** Name = aluminum sulfate
- Paragraph 3: **Find:** Chemical formula
- Paragraph 4: The rule is: write each ion with its charge, then balance total positive and negative charge to zero. Aluminum forms Al3+, and sulfate is SO42-.
- Paragraph 5: The least common total charge is 6, so use 2 aluminum ions for +6 and 3 sulfate ions for -6.
- Paragraph 6: Because sulfate is a polyatomic ion used more than once, keep it in parentheses: **Al2(SO4)3**.
