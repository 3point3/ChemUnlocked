Introductory General Chemistry  ·  Unit 05

Nomenclature

Naming compounds is chemistry's shared language. Learn the rules chemists use to name ionic compounds, molecular compounds, and acids — so you can read and write chemical names with confidence.

5.1 What Is Nomenclature?

Nomenclature means the rules for naming things. In chemistry, nomenclature rules let chemists name millions of compounds in a clear, organized way.

When you see a chemical name, you should be able to figure out what atoms are in the compound. When you see a formula, you should be able to write the name. Both skills matter.

🔑 Big Idea
There are three main types of compounds with their own naming rules: ionic compounds, molecular compounds, and acids.

5.2 Ionic Compounds — Metal + Nonmetal

An ionic compound is made of a metal and a nonmetal. The metal gives electrons to the nonmetal. This creates ions that attract each other.

How to name them — two simple steps:

Step 1 — Write the metal name first.

Write the name of the metal exactly as it appears on the periodic table. Do not change it.

Step 2 — Write the nonmetal name with "–ide" at the end.

Drop the end of the nonmetal's name and add –ide. For example: chlorine → chloride, oxygen → oxide, sulfur → sulfide, nitrogen → nitride, fluorine → fluoride, bromine → bromide, iodine → iodide.

5.3 Transition Metals — Roman Numerals

Some metals can form more than one type of ion. These are mainly transition metals like iron (Fe), copper (Cu), and lead (Pb).

When a metal can form more than one charge, you must add a Roman numeral in parentheses after the metal name. The Roman numeral shows the charge of the metal ion in that compound.

Pattern [Metal name](Roman numeral) [Nonmetal]-ide
Transition-Metal Compound Names with Roman Numerals
FormulaMetal ChargeName
FeCl₂Fe²⁺iron(II) chloride
FeCl₃Fe³⁺iron(III) chloride
CuOCu²⁺copper(II) oxide
Cu₂OCu⁺copper(I) oxide
FeOFe²⁺iron(II) oxide
Fe₂O₃Fe³⁺iron(III) oxide
💡 How to Find the Roman Numeral
Use the total charge of the nonmetal to figure out the metal charge. In FeCl₂: each Cl has a −1 charge. Two Cl⁻ ions = −2 total. The iron must be +2 to balance. So we write iron(II).
▸

Metals that always form one charge (like Na⁺, K⁺, Mg²⁺, Al³⁺, Ca²⁺) do not need Roman numerals. Only use them when the metal can have more than one charge.

💡 Pattern Tip — -ate vs. -ite
Ions ending in -ate have more oxygen. Ions ending in -ite have fewer oxygen. Example: nitrate (NO₃⁻) has 3 oxygens; nitrite (NO₂⁻) has 2 oxygens.

When naming a compound with a polyatomic ion, use the name of the ion directly. Do not add "-ide."

Examples: Ca(OH)₂ = calcium hydroxide  ·  NH₄NO₃ = ammonium nitrate  ·  Na₂SO₄ = sodium sulfate

5.5 Molecular Compounds — Nonmetal + Nonmetal

A molecular compound is made of two nonmetals. Nonmetals can bond in many different ratios. So we use prefixes to show exactly how many atoms of each element are in the molecule.

Step 1 — Name the first element with a prefix (skip "mono" for the first element).

Use the prefix that matches the subscript. If there is only 1 atom of the first element, skip "mono."

Step 2 — Name the second element with a prefix + –ide ending.

Always use a prefix for the second element. Change the ending to –ide. If the prefix ends in a or o and the element name starts with a vowel, drop the vowel from the prefix (e.g., tetra- + oxide = tetroxide, not tetraoxide).

▸

The biggest mistake: using prefixes for ionic compounds. Only use prefixes for molecular compounds (nonmetal + nonmetal). The subscripts in ionic compounds come from the ion charges — they are not a choice.

5.6 Acids — Binary Acids

An acid is a compound that starts with hydrogen and dissolves in water to release H⁺ ions.

A binary acid contains only two elements: hydrogen and one other nonmetal (no oxygen).

Binary Acid Naming Pattern hydro- + [element root] + -ic acid
Step 1 — Add the prefix "hydro-"

All binary acids start with "hydro-"

Step 2 — Write the root of the nonmetal's name

For chlorine → chlor, bromine → brom, iodine → iod, sulfur → sulfur, fluorine → fluor

Step 3 — Add "-ic acid" at the end

5.7 Acids — Oxyacids

An oxyacid contains hydrogen, oxygen, and one other element. You name oxyacids based on the polyatomic ion inside the acid.

💡 The Rule
If the ion ends in –ate → the acid ends in –ic acid
If the ion ends in –ite → the acid ends in –ous acid

5.8 How to Tell Compound Types Apart

The key to naming correctly is identifying the type of compound first. Use this decision guide:

Does the formula start with H?

→ It is an acid. Does it contain oxygen? If no: binary acid (hydro-…-ic acid). If yes: oxyacid (use ion name, change ending).

Is the first element a metal?

→ It is an ionic compound. Is the metal always one charge (like Na, K, Ca, Mg, Al)? If yes: just write the name + nonmetal-ide. If no (transition metal): add Roman numeral.

Are both elements nonmetals (and formula does not start with H)?

→ It is a molecular compound. Use prefixes for both elements. Add -ide to the second element.

🔑 Quick Checklist
Metal + nonmetal = ionic (no prefixes)  ·  Nonmetal + nonmetal = molecular (use prefixes)  ·  Starts with H = acid (different rules)

Ex 1 Naming a Simple Ionic Compound

Name: Na₂S

Identify the elements

Na = sodium (a metal)  ·  S = sulfur (a nonmetal)

Identify the compound type

Metal + nonmetal = ionic compound. Na always has a +1 charge — no Roman numeral needed.

Write the metal name, then the nonmetal with "-ide"

Sodium → sodium (unchanged). Sulfur → sulfide (–ide added).

Answer
sodium sulfide

Ex 2 Naming an Ionic Compound with a Transition Metal

Name: Fe₂O₃

Identify the elements

Fe = iron (transition metal, variable charge)  ·  O = oxygen (nonmetal, ion is O²⁻)

Find the iron charge using balance

Total negative charge: 3 oxide ions × (−2) = −6. Two iron atoms must supply +6 total. +6 ÷ 2 = +3 per iron atom. Roman numeral = III.

Write the name with Roman numeral

Iron(III) + oxide

Answer
iron(III) oxide

Ex 3 Naming a Compound with a Polyatomic Ion

Name: Cr₂(SO₄)₃

Identify the ions

SO₄²⁻ = sulfate (a polyatomic ion, charge = −2). Cr = chromium (transition metal, variable charge).

Find the chromium charge

3 sulfate ions × (−2) = −6 total. 2 chromium atoms = +6 total. +6 ÷ 2 = +3. Roman numeral = III.

Write the name

Use the polyatomic ion name directly — do not add "-ide."

Answer
chromium(III) sulfate

Ex 4 Naming a Molecular Compound

Name: N₂O₄

Identify the elements and subscripts

N (nitrogen) — subscript 2. O (oxygen) — subscript 4. Both are nonmetals = molecular compound.

First element: use prefix, skip "mono"

2 nitrogen atoms = dinitrogen

Second element: prefix + "-ide", drop vowel clash

4 oxygen atoms = tetra- + oxide. "Tetraoxide" → drop the "a" → tetroxide. Add to name.

Answer
dinitrogen tetroxide

Ex 5 Naming a Binary Acid

Name: HBr (dissolved in water)

Identify the type

Starts with H. No oxygen. = binary acid.

Apply the pattern: hydro- + element root + -ic acid

Bromine → brom. Pattern gives: hydro + brom + ic acid = hydrobromic acid.

Answer
hydrobromic acid

Ex 6 Naming an Oxyacid

Name: H₂SO₃

Identify the type

Starts with H. Contains oxygen. = oxyacid.

Identify the polyatomic ion inside

SO₃²⁻ = sulfite (ends in –ite).

Apply the rule: –ite → –ous acid

Sulfite → sulfur + ous → sulfurous acid.

Answer
sulfurous acid
